# Unity Shader Project

This project is focused on the **creation and experimentation of shaders in Unity**.

## Description

This repository contains different shader experiments and developments made in Unity, focusing on:
- Custom shaders and materials
- Visual effects
- Lighting, color, and shape experimentation
- Shader optimization and rendering techniques

The main goal of this project is to learn, prototype, and improve skills in shader development within the Unity engine.

## Technologies

- Unity
- ShaderLab / HLSL
- GPU rendering
